    Aanvullende data met embedding

    Deze WFS ondersteund dynamisch invoegen van gerelateerde data. Dat gebeurt door ?embed=... of ?expand=... aan de URL mee te geven. Mogelijke waarden:

    {% if embeddable_fields %} {% for expand in embeddable_fields %} {% endfor %}
    RelatieTabelOmschrijving
    {{ expand.name }} {{ expand.relation_id }} {{ expand.description|default:"" }}
    {% endif %} {% if expand_only_fields %}

    De volgende many-to-many relaties kunnen alleen met ?expand=... ingevoegd worden:

    {% for expand in expand_only_fields %} {% endfor %}
    RelatieTabelOmschrijving
    {{ expand.name }} {{ expand.relation_id }} {{ expand.description|default:"" }}
    {% endif %} {% if expandable_fields|length > 1 and 'embed=' not in connect_url %}

    Je kan meerdere waarden met een comma scheiden, bijvoorbeeld:

    {{ connect_url }}{% if '?' in connect_url %}&{% else %}?{% endif %}embed={{ expandable_fields.0.name }},{{ expandable_fields.1.name }}

    {% endif %} {% endif %}
